# TODOAPP-NodeJS

A backend REST API for a Task Management application built using **Node.js**, **Express.js**, and **MongoDB** with **Mongoose**.

* **Full CRUD Lifecycle:** Create, Read, Update, and Delete tasks seamlessly.
---

## 🛠️ Tech Stack

* **Runtime Environment:** Node.js
* **Backend Framework:** Express.js
* **Database Object Modeling:** Mongoose
* **Database:** Local MongoDB
* **API Testing Client:** Postman
